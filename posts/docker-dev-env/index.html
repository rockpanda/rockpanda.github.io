зңӢдәҶдёӢзҪ‘дёҠе…¶д»–дәәеҶҷзҡ„dockerејҖеҸ‘зҺҜеўғжҗӯе»әпјҢиҰҒд№ҲжҳҜеңЁubuntuдёӢжҗӯе»әпјҢиҰҒд№Ҳе°ұжҳҜдҪҝз”Ёе®ҳж–№иҜҙжҳҺзҡ„build docker-devй•ңеғҸзҡ„ж–№ејҸдёҖжӯҘжӯҘжҗӯе»әзҡ„пјҢз”ҡжҳҜз№ҒзҗҗпјҢdocker hubдёҠжңүдёҖдёӘdockerе®ҳж–№жҺЁеҮәзҡ„dockercore/dockerй•ңеғҸпјҢе…¶е®һиҝҷе°ұжҳҜе®ҳзҪ‘дёҠжүҖиҜҙзҡ„docker-devй•ңеғҸпјҢдёҚиҝҮд»ҘеүҚзҡ„йӮЈдёӘdeprecatedдәҶпјҢдҪҝз”Ёзӣ®еүҚиҝҷдёӘй•ңеғҸжҗӯе»әdockerејҖеҸ‘зҺҜеўғжҳҜжңҖеҝ«жҚ·зҡ„дәҶгҖӮ

жғіиҰҒдҝ®ж”№dockerжәҗз Ғе’ҢеҒҡdockerе®ҡеҲ¶ејҖеҸ‘зҡ„еҗҢеӯҰеҸҜд»ҘеҸӮиҖғдёӢгҖӮ

  • е®ҳж–№жҢҮеҜјж–ҮжЎЈ
  • и®ҫзҪ®dockerејҖеҸ‘зҺҜеўғ

dockerзҡ„зј–иҜ‘е®һиҙЁдёҠжҳҜеңЁdockerе®№еҷЁдёӯиҝҗиЎҢdockerгҖӮ

еӣ жӯӨеңЁжң¬ең°зј–иҜ‘dockerзҡ„еүҚжҸҗжҳҜйңҖиҰҒе®үиЈ…дәҶdockerпјҢиҝҳйңҖиҰҒз”ЁgitжҠҠд»Јз ҒpullдёӢжқҘгҖӮ

еҲӣе»әеҲҶж”Ҝ

дёәдәҶж–№дҫҝд»ҘеҗҺз»ҷdockerжҸҗдәӨжӣҙж”№пјҢжҲ‘д»¬д»Һdockerе®ҳж–№forkдёҖдёӘеҲҶж”ҜгҖӮ

git clone https://github.com/rootsongjc/docker.git
git config --local user.name "Jimmy Song"
git config --local user.email "rootsongjc@gmail.com"
git remote add upstream https://github.com/docker/docker.git
git config --local -l
git remote -v
git checkout -b dry-run-test
touch TEST.md
vim TEST.md
git status
git add TEST.md
git commit -am "Making a dry run test."
git push --set-upstream origin dry-run-test

з„¶еҗҺе°ұеҸҜд»ҘеңЁdry-run-testиҝҷдёӘеҲҶж”ҜдёӢе·ҘдҪңдәҶгҖӮ

й…ҚзҪ®dockerејҖеҸ‘зҺҜеўғ

е®ҳзҪ‘дёҠиҜҙйңҖиҰҒе…Ҳжё…з©әиҮӘе·ұз”өи„‘дёҠе·Іжңүзҡ„е®№еҷЁе’Ңй•ңеғҸгҖӮ

dockerејҖеҸ‘зҺҜеўғжң¬иҙЁдёҠжҳҜеҲӣе»әдёҖдёӘdockerй•ңеғҸпјҢй•ңеғҸйҮҢеҢ…еҗ«дәҶdockerзҡ„жүҖжңүејҖеҸ‘иҝҗиЎҢзҺҜеўғпјҢжң¬ең°д»Јз ҒйҖҡиҝҮжҢӮиҪҪзҡ„ж–№ејҸж”ҫеҲ°е®№еҷЁдёӯиҝҗиЎҢпјҢдёӢйқўиҝҷжқЎе‘Ҫд»ӨдјҡиҮӘеҠЁеҲӣе»әиҝҷж ·дёҖдёӘй•ңеғҸгҖӮ

еңЁdry-run-testеҲҶж”ҜдёӢжү§иЎҢ

make BIND_DIR=. shell

иҜҘе‘Ҫд»ӨдјҡиҮӘеҠЁзј–иҜ‘дёҖдёӘdockerй•ңеғҸпјҢFrom debian:jessieгҖӮиҝҷдёҖжӯҘдјҡдёҠзҪ‘дёӢиҪҪеҫҲеӨҡдҫқиө–еҢ…пјҢйҖҹеәҰжҜ”иҫғж…ўгҖӮеҰӮжһңзҝ»дёҚдәҶеўҷзҡ„иҜқиӮҜе®ҡйғҪдјҡеӨұиҙҘгҖӮеӣ дёәйңҖиҰҒдёӢиҪҪзҡ„иҪҜд»¶е’Ңе®үиЈ…еҢ…йғҪжҳҜеңЁеӣҪеӨ–жңҚеҠЎеҷЁдёҠпјҢдёҚзҝ»еўҷж №жң¬е°ұдёӢиҪҪдёҚдёӢжқҘпјҢдёәдәҶдёҚз”Ёиҝҷд№Ҳйә»зғҰпјҢжҺЁиҚҗзӣҙжҺҘдҪҝз”Ёdockerе®ҳж–№зҡ„dockercore/dockerй•ңеғҸпјҢд№ҹдёҚз”Ёд»ҘеүҚзҡ„docker-devй•ңеғҸпјҢйӮЈдёӘйҖ е°ұеәҹејғдәҶгҖӮиҝҷдёӘй•ңеғҸеӨ§е°Ҹжңү2.31GгҖӮ

docker pull dockercore/docker

дҪҝз”Ёж–№жі•и§Ғ docker hub

з„¶еҗҺе°ұеҸҜд»Ҙиҝӣе…ҘеҲ°е®№еҷЁйҮҢ

docker run --rm -i --privileged -e BUILDFLAGS -e KEEPBUNDLE -e DOCKER_BUILD_GOGC -e DOCKER_BUILD_PKGS -e DOCKER_CLIENTONLY -e DOCKER_DEBUG -e DOCKER_EXPERIMENTAL -e DOCKER_GITCOMMIT -e DOCKER_GRAPHDRIVER=devicemapper -e DOCKER_INCREMENTAL_BINARY -e DOCKER_REMAP_ROOT -e DOCKER_STORAGE_OPTS -e DOCKER_USERLANDPROXY -e TESTDIRS -e TESTFLAGS -e TIMEOUT -v "/Users/jimmy/Workspace/github/rootsongjc/docker/bundles:/go/src/github.com/docker/docker/bundles" -t "dockercore/docker:latest" bash
